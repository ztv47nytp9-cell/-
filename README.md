# Victor Alpha 0.18.4 Integrated Flow

기동방제계 업무 흐름 반영 통합 수정판입니다.

## 핵심 변경

- 일반등록/긴급기록 버튼형 UI 적용
- 일반등록 출고/입고 선택
- 출고/입고 저장 시 재고 즉시 반영 및 이력 자동 생성
- 긴급기록은 제목/보관 없이 미반영 저장
- 긴급기록 반영 시 재고 차감 + 출고이력 자동 생성 + 긴급기록 완료 상태 유지
- 홈 문구 정리
- 창고 상세 상단에서 위치/담당/연락처 삭제, 중요 메모만 표시
- 지세포창고 명칭을 한국석유공사 거제지사 창고로 정리
- 창고/자재목록/자재/장비 신규 추가
- 추가 항목이 재고/등록/이력에 반영
- 장비 기본 목록 추가
