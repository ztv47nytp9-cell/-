# Victor Alpha 0.19.0e UX Beta

0.19.0d 안정판을 보존하고 홈·메모·이력·이동 편의성을 확장한 베타 버전입니다. 기존 데이터는 자동 이전됩니다.

## 홈 화면

- 긴급기록·출고·입고 바로등록
- 오늘 출고·입고·재고수정 건수와 즉시 필터 이동
- 최근 방문한 창고·장비·이력 이어가기
- 최근 활동을 이력·메모 탭으로 통합

## 사용성

- 하단 메뉴를 홈·보관·등록·이력 4개로 정리
- 메모 작성·전체보기·수정을 홈에서 연결
- 이력 기간·종류 필터
- 휴대전화 뒤로가기 상태 복원
- 설정 가능한 햅틱 반응
- 시에라 블루 메뉴와 새 마스커블 아이콘

## 수정

- 미반영 긴급기록 상세 화면의 반환 오류 수정
