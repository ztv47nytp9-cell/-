# Victor Alpha 0.19.0c Stable

긴급상황 현장 기록과 사후 정산 흐름을 강화한 안정화 버전입니다. 기존 0.19.0b 및 이전 데이터는 자동 이전됩니다.

## 긴급기록 업무 흐름

1. 현장에서는 제목과 보관장소 없이 사용 자재·수량만 미반영 저장
2. 이력의 미반영 기록에서 사고명, 보관장소, 날짜, 메모, 자재, 수량 보완
3. 재고 확인 후 한 번만 출고 반영하고 완료 처리

## 신뢰성 개선

- 미반영 기록과 완료 기록을 이중 생성하지 않음
- 이미 완료된 기록은 다시 반영할 수 없음
- 반영 중 저장 실패 시 차감 수량을 원상 복구
- 완료 기록 삭제 시 재고를 이전 상태로 복구
- 기존 무한 로딩 방지, 캐시 복구, 손상 데이터 방어 유지
