# Victor Alpha 0.19.0d Stable

보관 상세의 신규 추가 기능을 프롬프트에서 버튼형 화면으로 개편한 버전입니다. 기존 0.19.0c 및 이전 데이터는 자동 이전됩니다.

## 화면 개선

- 점 3개 메뉴: 백업, 복원, 앱 정보, 전체 초기화만 유지
- 업데이트 내역을 앱 정보 화면에 통합
- 자재 추가와 자재목록 추가를 큰 선택 버튼으로 구분
- 장비 추가와 장비목록 추가를 큰 선택 버튼으로 구분
- 모든 세부 입력을 앱 내부 폼에서 처리

## 업무 기능

- 신규 자재 등록 시 해당 창고 초기 수량 입력 가능
- 초기 수량이 있으면 재고수정 이력을 자동 생성
- 새 장비목록을 데이터에 영구 저장
- 장비 상태를 정상, 점검필요, 수리중, 사용불가 중 선택
