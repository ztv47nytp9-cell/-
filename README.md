# Victor Alpha 0.19.0f Resource Beta

기존 0.19.0e 로컬 데이터 키와 핵심 기능을 유지하면서 보관 조회와 자원 입력을 개선한 베타 버전입니다. 공용 계정·권한·DB 구조는 포함하지 않습니다.

## 보관 화면

- 창고별 보기, 전체 자재, 전체 장비 전환
- 전체 장비의 장비명·규격·모델명·메모·보관창고 검색
- 장비 카테고리·창고 필터
- 전체 장비에서 상세 확인 및 수정

## 단순 입력

- 품목명, 규격, 수량, 보관창고, 메모, 사진 중심
- 사진 자동 축소 저장
- 기존 세부 장비 정보는 삭제하지 않고 상세에 보조 표시

## iPhone PWA 피드백

- navigator.vibrate 지원 여부 확인
- 미지원 환경에서 햅틱 설정 비활성화
- 버튼 눌림 애니메이션과 성공·오류 토스트 제공
- 재고 부족 품목과 부족량 표시

## 유지

- 기존 창고별 조회, 등록, 이력, 미반영 기록 및 저장 데이터
