# Victor Alpha 0.19.0j Fast Survey Beta

- 창고·분류 범위를 고르는 신속 재고실사
- 0·현재 수량·직접 입력과 12개 단위 일괄 입력
- 차이가 있는 품목만 검토 후 선택 범위에 반영
- 장비의 현재 부속품 목록을 관리이력과 분리
- 등록 작성 중 자동복구(사진 제외)
- 유처리제 리터·18L 캔 병행 표시
- 차량·방제정의 가장 최근 누적 이력 수정·삭제

- 등록 화면 자재·장비 분리 및 장비 사용이력 저장
- 장비 수리·정비·부속품·이동이력
- 브라우저 프롬프트 없는 앱 내부 입력양식
- 홈 전체 보관량과 자주 쓰는 기능
- 창고 간 자재 이동
- 읽기 전용 Victor 공유자료
- 사진 촬영·사진 선택 분리와 저장공간 보호

- 홈에서 자재·장비·창고 통합 찾기
- 직전 등록의 창고·품목 참고
- 수량 +1·+5·+10 빠른 입력

기존 0.19.0e 로컬 데이터 키와 핵심 기능을 유지하면서 보관 조회와 자원 입력을 개선한 베타 버전입니다. 공용 계정·권한·DB 구조는 포함하지 않습니다.

## 보관 화면

- 창고별 보기, 전체 자재, 전체 장비 전환
- 전체 장비의 장비명·규격·모델명·메모·보관창고 검색
- 장비 카테고리·창고 필터
- 전체 장비에서 상세 확인 및 수정

## 단순 입력

- 품목명, 규격, 수량, 보관창고, 메모, 사진 중심
- 사진 자동 축소 저장
- 기존 세부 장비 정보는 삭제하지 않고 상세에 보조 표시

## iPhone PWA 피드백

- navigator.vibrate 지원 여부 확인
- 미지원 환경에서 햅틱 설정 비활성화
- 버튼 눌림 애니메이션과 성공·오류 토스트 제공
- 재고 부족 품목과 부족량 표시

## 유지

- 기존 창고별 조회, 등록, 이력, 미반영 기록 및 저장 데이터
