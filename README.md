# Victor Alpha 0.18.6 Integrated Storage Summary

0.18.4, 0.18.4a, 0.18.5 기능을 합친 통합판입니다.

## 포함 기능

- 0.18.4 통합 흐름
  - 일반등록/긴급기록 큰 버튼
  - 출고/입고 선택
  - 긴급기록 미반영 → 재고반영 → 출고이력 생성
  - 창고/자재목록/자재/장비 신규 추가
- 0.18.4a 장비 Hotfix
  - 실제 장비표 기준 기본 장비 목록
  - 장비 추가 후 보관장소 장비 탭 반영
- 0.18.5 재고수정 이력
  - 재고 직접 수정 시 변경사유 입력
  - 변경 전/후 수량 이력 생성
